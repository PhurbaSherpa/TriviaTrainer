# Trivia Trainer

A web application built with Node.js and React, it allows user the ability to train themselves with trivia questions to become better at them.

## Features

<h2>You can crate your own account and keep your scores of past quizzes</h2>
<img src="https://i.imgur.com/vEdxsUJ.png" width='40%'/>
<h2>You can go back to past quizzes to review your answers</h2>
<img src="https://i.imgur.com/cjQzvyd.png" width='40%'/>
<h2> Live score update as you're taking the quiz. </h2>

## Setup

Fork the repo and then git clone that repo

```
git clone https://github.com/your_github/TriviaTrainer
```

then

```
npm install
```

Then run it

```
npm run start
```

If you want to make additions to it run it in dev

```
npm run start-dev
```

## Testing

* includes testing for routes and react components
* put an x in front of describe to block a test

run the following code:

```
npm run test
```
